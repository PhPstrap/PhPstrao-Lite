# PhPstrap Lite

PhPstrap Lite is a lightweight PHP and Bootstrap starter kit for building fast static-style websites without a database or complicated CMS setup.

It includes reusable page sections, shared header and footer includes, per-page SEO meta, a flat-file blog/article system, a contact form, sitemap support, and a simple configuration file. It is ideal for landing pages, small business websites, affiliate sites, guides, documentation pages, and simple content projects.
